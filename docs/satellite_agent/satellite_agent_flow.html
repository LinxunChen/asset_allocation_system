卫星 Agent 主链路

基于最新代码抽象出的系统骨架图，突出三池流转与复盘闭环

关注名单 = 系统持续跟踪的标的范围
候选池 = 短期记忆层，默认静默记录
确认池 = 正式机会，会推送飞书正式卡
持仓管理 = 持仓后的统一退出管理

阅读方式：先看关注名单内如何发现机会，再看三池流转，最后看复盘闭环。说明：图中的成交、退出、结算均为策略后验评估口径，不代表真实自动下单；其中“窗口到期结算”用于给复盘提供统一截止时间，不等于真实卖出。

归池逻辑

if 不值得继续跟踪 -> 丢弃 elif 值得跟踪，但暂不适合正式出手 -> 进入候选池 elif 消息强，盘面也支持，且适合执行 -> 进入确认池 if 候选近期再次增强 -> 候选升级到确认池

flowchart TD
    subgraph Discovery["阶段一：机会发现"]
        direction TB
        W["关注名单"] --> E["关注名单内的事件发现"]
        W --> S["关注名单内的行情扫描"]
        E --> R{"归池决策"}
        S --> R
    end

    subgraph Lifecycle["阶段二：三池与交易生命周期"]
        direction TB

        R -- "先跟踪，暂不正式出手" --> C["第一池：候选池<br/>跨轮次短期记忆（默认 72h）<br/>默认静默记录"]
        R -- "消息强，盘面也支持" --> F["第二池：确认池<br/>正式机会<br/>推送飞书正式卡"]

        C -. "重点候选时，可选发送观察卡" .-> CL["观察卡（可选）<br/>可选推送飞书观察卡"]
        C -. "后续出现新催化，可升级" .-> F

        F --> O["等待成交"]
        O -- "触发买点" --> H["持仓中"]
        O -. "触发失效价" .-> X_PRICE["本次机会结束<br/>失效未成交"]
        O -. "窗口到期仍未成交" .-> X_TIMEOUT["本次机会结束<br/>窗口内未成交"]

        H --> P["第三池：持仓管理<br/>统一处理止盈 / 失效 / 窗口到期"]
        P -- "达到目标 / 利润保护" --> EXIT_WIN["退出操作<br/>利润保护退出<br/>推送飞书退出卡"]
        P -- "触发失效价" --> EXIT_LOSS["退出操作<br/>失效价退出<br/>推送飞书退出卡"]
        P -- "窗口到期" --> EXIT_TIME["退出操作<br/>窗口到期结算<br/>只落盘复盘，不推送"]
    end

    subgraph Review["阶段三：复盘闭环"]
        direction LR
        C --> DB[("后验数据库")]
        H --> DB
        X_PRICE --> DB
        X_TIMEOUT --> DB
        EXIT_WIN --> DB
        EXIT_LOSS --> DB
        EXIT_TIME --> DB

        DB --> RUN["运行复盘"]
        DB --> HIST["历史复盘<br/>30d / 90d / 路径质量 / 候选转化"]
    end

    Discovery ==> Lifecycle
    Lifecycle ==> Review
